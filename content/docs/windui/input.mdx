## Input Element

Input fields allow users to enter text data. They support both single-line and multi-line (textarea) inputs.

<Tabs defaultValue="small">
    <TabsList>
        <TabsTrigger value="small">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="small">
```luau
local Input = Tab:Input({
    Title = "Username",
    Callback = function(text)
        print("Input:", text)
    end
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
local Input = Tab:Input({
    Title = "Username",
    Desc = "Enter your username", -- optional
    Icon = "user", -- lucide icon or "rbxassetid://". optional
    Type = "Default", -- "Default" or "Textarea". optional
    Placeholder = "Type here...", -- placeholder text. optional
    Value = "", -- initial value. optional
    Locked = false, -- disable input. optional
    Flag = "my_input", -- for config saving. optional
    Callback = function(text)
        print("Text entered:", text)
    end
})
```
    </TabsContent>
</Tabs>

## Input Variants

### Textarea Input

```luau
Tab:Input({
    Title = "Description",
    Type = "Textarea",
    Placeholder = "Enter detailed description...",
    Callback = function(text)
        print("Description:", text)
    end
})
```

### Input with Icon

```luau
Tab:Input({
    Title = "Icon Input",
    Icon = "key",
    Callback = function(v)
        print("Entered", v)
    end
})
```
