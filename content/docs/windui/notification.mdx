## Notification System

Notifications are temporary messages displayed to inform users about events or status changes. They automatically disappear after a set duration.

<Tabs defaultValue="small">
    <TabsList>
        <TabsTrigger value="small">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="small">
```luau
WindUI:Notify({
    Title = "Success",
    Content = "Operation completed successfully",
    Duration = 3
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
WindUI:Notify({
    Title = "Success",
    Content = "Operation completed successfully",
    Icon = "check-circle", -- lucide icon or "rbxassetid://". optional
    Duration = 3, -- time in seconds. optional
    Color = Color3.fromRGB(100, 255, 100), -- custom color. optional
    TextColor = Color3.fromRGB(255, 255, 255), -- custom text color. optional
})
```
    </TabsContent>
</Tabs>
