## Key System Overview

The Key System provides access control to your WindUI window. When enabled, users must enter a valid key before accessing the interface. This is useful for restricting access to premium, admin-only, or exclusive scripts.

## Basic Usage

<Tabs defaultValue="simple">
    <TabsList>
        <TabsTrigger value="simple">Simple</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>

```luau
local Window = WindUI:CreateWindow({ -- creating window from /docs/windui/window
    Title = "Admin Panel",
    KeySystem = {
        KeyValidator = function(key)
            return key == "admin123"
        end,
    }
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
local Window = WindUI:CreateWindow({ -- creating window from /docs/windui/window
    Title = "Premium Tool",
    KeySystem = {
        Title = "Access Required",
        Note = "Get your key at discord.gg/example",
        
        KeyValidator = function(key) -- your key validation logic
            local validKeys = {"key1", "key2", "key3"}
            return table.find(validKeys, key) ~= nil
        end,
        
        SaveKey = true,
        URL = "https://discord.gg/example", -- link to get keys
        
        Thumbnail = {
            Image = "rbxassetid://1234567890",
            Title = "Premium Member",
        }
    }
})
```
    </TabsContent>
</Tabs>

The KeySystem must be located inside the [Window](/docs/windui/window) configuration table passed to `WindUI:CreateWindow()`

## Examples

### Simple Key Validation

<DynamicCodeBlock className="m-0" lang="luau" code={`local Window = WindUI:CreateWindow({
    Title = "My Script",

### Multiple Valid Keys

<DynamicCodeBlock className="m-0" lang="luau" code={`local Window = WindUI:CreateWindow({
    Title = "Admin Panel",

## Save Key Behavior

When `SaveKey` is enabled:

1. User enters key in the dialog
2. Key is validated using `KeyValidator`
3. If valid, key is saved to `[Folder]/[player.Name].key`
4. On future runs, saved key is loaded and validated automatically

<DynamicCodeBlock className="m-0" lang="luau" code={`local Window = WindUI:CreateWindow({
    Title = "App",
    Folder = "MyApp", -- key saved to "MyApp/[player.Name].key"
    KeySystem = {
        KeyValidator = function(key)
            return key == "example"
        end,
        SaveKey = true -- enables persistence
    }
})`} />
