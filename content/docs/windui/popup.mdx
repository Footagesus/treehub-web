## Popup Component

Popups are standalone windows that display information with action buttons. They're useful for announcements, confirmations, and important messages.

<Tabs defaultValue="small">
    <TabsList>
        <TabsTrigger value="small">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="small">
```luau
local Popup = WindUI:Popup({
    Title = "Welcome",
    Content = "Welcome to our application!",
    Buttons = {
        {
            Title = "Get Started",
            Callback = function()
                print("Getting started...")
            end
        }
    }
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
local Popup = WindUI:Popup({
    Title = "Important Message",
    Icon = "alert", -- lucide icon or "rbxassetid://". optional
    Content = "This is important information",
    Buttons = {
        {
            Title = "Continue",
            Icon = "arrow-right", -- optional
            Variant = "Primary", -- "Primary", "Secondary", "Tertiary". optional
            Callback = function()
                print("Proceeding...")
            end
        },
        {
            Title = "Cancel",
            Icon = "x", -- optional
            Variant = "Secondary", -- optional
            Callback = function()
                print("Cancelled")
            end
        }
    }
})
```
    </TabsContent>
</Tabs>

## Popup Methods

### Close Popup

Close and hide the popup.

```luau
Popup:Close()
```

## Differences: Popup vs Dialog

- **Popup**: Standalone windows with title and full styling, for announcements
- **Dialog**: Modal prompts that block window interaction, for confirmations
- **Notification**: Temporary messages that auto-dismiss, for feedback
