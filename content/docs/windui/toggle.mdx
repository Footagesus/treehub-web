## Toggle Element

Toggles are switch elements that represent binary states (on/off, true/false). They're ideal for enabling/disabling features.

<Tabs defaultValue="small">
    <TabsList>
        <TabsTrigger value="small">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="small">
```luau
local Toggle = Tab:Toggle({
    Title = "Enable Feature",
    Callback = function(state)
        print("Toggle state:", state)
    end
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
local Toggle = Tab:Toggle({
    Title = "Enable Feature",
    Desc = "Toggle description", -- optional
    Icon = "power", -- lucide icon or "rbxassetid://". optional
    Value = false, -- initial state. optional
    Type = "Toggle", -- "Toggle" or "Checkbox". optional
    Color = Color3.fromRGB(100, 200, 100), -- custom color. optional
    Locked = false, -- disable toggle. optional
    Flag = "my_toggle", -- for config saving. optional
    Callback = function(state)
        print("State changed:", state)
    end
})
```
    </TabsContent>

## Toggle Variants

### Simple Toggle

```luau
Tab:Toggle({
    Title = "Feature Enabled",
    Value = false,
    Callback = function(state)
        print("Feature enabled:", state)
    end
})
```

### Checkbox Style Toggle

```luau
Tab:Toggle({
    Title = "Accept Terms",
    Type = "Checkbox",
    Value = false,
    Callback = function(state)
        print("Terms accepted:", state)
    end
})
```

### Toggle with Icon

```luau
Tab:Toggle({
    Title = "Enable Notifications",
    Icon = "bell",
    Value = true,
    Callback = function(state)
        print("Notifications:", state and "Enabled" or "Disabled")
    end
})
```

### Locked Toggle

```luau
local PremiumToggle = Tab:Toggle({
    Title = "Premium Feature",
    Locked = true,
    Value = false,
    Callback = function(state)
        print("Premium feature:", state)
    end
})
```
