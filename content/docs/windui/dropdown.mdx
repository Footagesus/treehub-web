## Dropdown Element

Dropdowns allow users to select from a list of predefined Values. They're useful for presenting multiple choices in a compact format.

<Tabs defaultValue="small">
    <TabsList>
        <TabsTrigger value="small">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
        <TabsTrigger value="advanced">Advanced</TabsTrigger>
    </TabsList>

    <TabsContent value="small">
```luau
local Dropdown = Tab:Dropdown({
    Title = "Select Value",
    Values = {
        "Option 1",
        "Option 2",
        "Option 3"
    },
    Callback = function(selected)
        print("Selected:", selected)
    end
})
```

## Dropdown Variants

### Simple Dropdown

```luau
Tab:Dropdown({
    Title = "Choose Theme",
    Values = {
        "Light",
        "Dark",
        "Auto"
    },
    Value = "Dark",
    Callback = function(selected)
        print("Theme:", selected)
    end
})
```

### Multi-Select Dropdown

```luau
Tab:Dropdown({
    Title = "Select Features",
    Desc = "Choose multiple features to enable",
    Values = {
        "Feature A",
        "Feature B",
        "Feature C",
        "Feature D"
    },
    Multi = true,
    AllowNone = true,
    Callback = function(selected)
        print("Selected features:", selected)
    end
})
```
